Модуль: Hibernate.
Лекция: Установка PostgreSQL в Docker-контейнере.

Что такое Docker?
Docker - это платформа, которая позволяет упаковать в контейнер приложение со всеми окружением и зависимостями в один
docker образ, который запускается в изолированной среде, не влияющей на основную операционную систему.

Виртуализация.
Виртуализация работает как отдельный компьютер со своей собственной операционной системой и виртуальным оборудованием.

Контейнеризация.
При контейнеризации виртуальная среда, запускается прямо из ядра основной ОС и НЕ визуализирует оборудование. Поэтому
контейнер может работать только в той же ОС, что и основная.

Запускаем Docker-контейнер с PostgreSQL внутри.

docker run --name postgres-database -p 5432:5432 -e POSTGRES_USER=suleymanov -e POSTGRES_PASSWORD=12345678 -e
POSTGRES_DB=suleymanov -d postgres:15.3.

docker run - просто запускает контейнер.
--name - задает название для контейнера.
-p - делает маппинг порта нашего компьютера с портом внутри контейнера.
-e - задает переменную окружения внутри контейнера.
-d - запуск контейнера в фоновом режиме.
postgres:15.3 - название и версия docker-образа.
